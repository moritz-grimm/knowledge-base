# Angebotskalkulation

## Übersicht

Die Angebotskalkulation ermittelt den Verkaufspreis basierend auf dem Einkaufspreis plus allen verbundenen Kosten wie Rabatten, Skonto, Steuern und Versandkosten. Sie folgt einem standardisierten schrittweisen Schema.

## Berechnungsschema

### Käuferseite

| Operation | Name                             | Erklärung                     |
| --------- | -------------------------------- | ----------------------------- |
|           | Listeneinkaufspreis              |                               |
| −         | Lieferantenrabatt                | Rabatt auf den Listenpreis    |
| =         | Zieleinkaufspreis                |                               |
| −         | Lieferantenskonto                | Rabatt für pünktliche Zahlung |
| =         | Bareinkaufspreis                 |                               |
| +         | Bezugskosten                     | Versand, Zoll                 |
| =         | **Bezugspreis / Einstandspreis** | Wahrer Preis aller Waren      |

### Verkäuferseite

| Operation | Name                           | Erklärung                                           |
| --------- | ------------------------------ | --------------------------------------------------- |
|           | Einstandspreis                 |                                                     |
| +         | Handlungsgemeinkosten          | Betriebskosten, Miete, Personal usw.                |
| =         | Selbstkostenpreis              |                                                     |
| +         | Gewinnzuschlag                 | Gewünschter Gewinn                                  |
| =         | Barverkaufspreis               |                                                     |
| +         | Kundenskonto                   | Erlaubt Kunden einen Rabatt bei pünktlicher Zahlung |
| =         | Zielverkaufspreis              |                                                     |
| +         | Kundenrabatt                   | Erlaubt Kunden einen Rabatt zu bekommen             |
| =         | Listenverkaufspreis netto      |                                                     |
| +         | Umsatzsteuer / MwSt.           | 19% in Deutschland                                  |
| =         | **Listenverkaufspreis brutto** | Finaler Preis für den Kunden                        |

## Notizen

- Skonto und Rabatt werden auf der Verkäuferseite wieder **hinzugefügt**, weil diese potenzielle Abzüge darstellen, die der Kunde einlösen kann – das muss durch den Listenverkaufspreis abgedeckt sein
- Die Handlungsgemeinkosten werden in der Regel als Prozentsatz des **Bezugspreises/Einstandspreises** angegeben, basierend auf den tatsächlichen Betriebskosten des Unternehmens
- Die Berechnung kann vorwärts (vom Einkaufspreis zum Verkaufspreis) oder rückwärts (vom Zielverkaufspreis zum erforderlichen Einkaufspreis) durchgeführt werden
