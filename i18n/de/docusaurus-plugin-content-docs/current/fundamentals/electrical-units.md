# Elektrische Einheiten - Leistung, Spannung und Stromstärke

## Die drei Kern Einheiten

| Symbol | Größe                 | Einheit    |
| ------ | --------------------- | ---------- |
| P      | Leistung (Power)      | Watt (W)   |
| U      | Spannung (Voltage)    | Volt (V)   |
| I      | Stromstärke (Current) | Ampere (A) |

## Das Leistungsdreieck

Die drei Einheiten sind alle in folgender Beziehung miteinander **P = U x I**. Wenn man die Einheit die man berechnen möchte abdeckt entsteht die Formel die man benötigt.

```text
      P
    -----
    U | I
```

- **P = U x I** (Leistung = Spannung x Stromstärke)
- **U = P / I** (Spannung = Leistung / Stromstärke)
- **I = P / U** (Stromstärke = Leistung / Spannung)

## Ohmsches Gesetz

Das Ohmsche Gesetz beschreibt den Widerstand **R** (gemessen in Ohms, Ω) und verbindet ihn mit Spannung und Strom:

- **U = R x I**
- **R = U / I**
- **I = U / R**

## Kombinierte Formeln

Das Verbinden der Formeln mit dem Ohmschen Gesetz bietet zusätzliche Möglichkeiten zur Berechnung der Leistung:

- **P = U² / R**
- **P = I² x R**

## Beispiele

**Beispiel 1:** Ein Gerät läuft mit 230 V und zieht 2 A. Wie hoch ist der Stromverbrauch (P)?

```text
P = U x I = 230 V x 2 A = 460 W
```

**Beispiel 2:** Eine 100 W Glühbirne arbeitet mit 230 V. Wie viel Strom (I) verbraucht sie?

```text
I = P / U = 100 W / 230 V ≈ 0.43 A
```

**Beispiel 3:** Ein Widerstand von 50 Ω führt 3 A. Wie groß ist der Spannungsabfall an ihm?

```text
U = R x I = 50 Ω x 3 A = 150 V
```
